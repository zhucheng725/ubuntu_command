# ubuntu_command
some commands <br>
1.tar <br>
```
$ sudo tar zcvf xxx.tar.gz files_path
```
<br>

2.delete files<br>
```
$ sudo rm path_files
```
<br>

3.delete files_dir<br>
```
$ sudo rmdir path_files
```
<br>

4.create files_dir<br>
```
$ sudo mkdir path_files
```
<br>

5.move A to B <br>
```
$ sudo mv A B
```
